## License terms

The AI-READI dataset is shared under a custom license that has been developed as part of the project. The full license terms are available [here](https://doi.org/10.5281/zenodo.10642459). It is recommended to read them first to ensure that your envisioned use of the AI-READI dataset fits within the requirements of the license.

## Overview

A brief summary of what you can and can't do with this version of the dataset are provided below.

Can:

- Use the dataset for type 2 diabetes related research
- Use the dataset for training AI/ML models and other computational models

Can't:

- Use the dataset for research unrelated to type 2 diabetes
- Attempt to identify study participants
