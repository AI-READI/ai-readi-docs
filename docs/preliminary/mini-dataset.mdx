## Mini version of the dataset

Since the AI-READI dataset is now quite large, we have created a mini version on popular demand that users can download first to explore the dataset and set up their pipelines.
This subset dataset contains data from 100 participants and 179.68 GB in size.

Further details about the subset dataset contents are available here. [here](https://fairhub.io/datasets/4).
