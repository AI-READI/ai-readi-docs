---

As per the [license terms](preliminary/license), if you use the AI-READI dataset or any subset of it, you must cite the following references:

1. ```
      AI-READI Consortium. "AI-READI: rethinking data collection, preparation and
      sharing for propelling AI-based discoveries in diabetes research and beyond."
      Nature metabolism (2024) https://doi.org/10.1038/s42255-024-01165-x
   ```
2. ```
      AI-READI Consortium. (2024). Flagship Dataset of Type 2 Diabetes from the
      AI-READI Project (2.0.0) [Data set]. FAIRhub.
      https://doi.org/10.60775/fairhub.2
   ```
